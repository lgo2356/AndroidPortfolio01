개요
---
- OpenAI API gpt-4-mini 모델을 활용한 AI와 대화하는 안드로이드 앱

기술 스택
---
- 언어: Kotlin
- 아키텍처: MVVM, SAA
- UI: Compose
- 네트워크: Retrofit
- DB: Room
- 비동기: Coroutine
- 로그인 인증: Firebase Authentication
- API 서버 : Firebase Cloud Functions, Firestore, Nodejs, OpenAI API
